Publicera på GitHub Pages:
1. Skapa ett repo eller öppna ditt befintliga repo.
2. Ladda upp alla filerna i den här mappen.
3. Se till att startsidan heter exakt index.html.
4. I GitHub: Settings > Pages.
5. Under Build and deployment välj:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
6. Spara.
7. Vänta tills länken visas under Pages.

Den här versionen:
- visar frågan högre upp på mobilen
- fortsätter där du slutade med hjälp av localStorage
- blandar frågorna om ingen sparad session finns
- innehåller 100 textbaserade frågor från urval 2023–2025
- hoppar över bild-, diagram- och tabellfrågor
